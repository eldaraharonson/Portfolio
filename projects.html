Popularity Check with Facebook

Overview:

The user logs in with their Facebook account and can use the following features:

  • Find the Facebook friend that has given the user the largest amount of likes on all of the user's likeable elements (posts, pictures, etc.). The friends pool can be filtered based on age and gender.
  • Find the number of members of any Facebook group that the user is a member of. The number of members may be filtered based on age and gender.

Development Information:

The app is a windows desktop app developed in C# using the .NET Framework and .NET WinForms. I developed it using OOP principles along with the Facade design pattern to help provide clear seperation between the UI and the logic.

Overview:

A console application developed in Java using OOP principles with utilization of Wireshark for debugging purposes.

Memory Game

Overview:

A game where the users are presented with a board of blank cards and need to find all of the matching pairs. In every turn the user picks a pair of cards, the cards' contents are shown for a few seconds and then are hidden again. Every time a matching pair is found the player is awarded a point. After all of the matching pairs have been found the player with the most points wins.
The game can be played using 2 players or 1 player against the CPU.

Development Information:

A windows desktop app developed in C# using the .NET Framework and .NET WinForms. I developed it using OOP principles.
